Una vez compilado el conjunto ocupando el archivo Makefile
debera ejecutar el archivo a.out generado para iniciar el
programa, los datos aceptados son [NS] como EJEX y [EO] como
EJEY siempre en mayusculas. Para poder moverse desde el punto
en diagonal debera elegir primero un dato del EJEY y luego del
EJEX, se hizo de esta manera tomando en cuenta los
puntos cardinales [Norte,Sur,Este,Oeste] y sus intermedios
[NorEste,NorOeste,SurEste,SurOeste] por lo que se necesitaria
ingresar primero un eje vertical y luego uno horizontal. Una vez
ingresado el string se mostrara el camino trazado y las coordenadas
finales.
